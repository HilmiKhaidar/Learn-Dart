# Modul 5 — Loops (Perulangan)

## 🎯 Tujuan
- Memahami konsep dasar perulangan.
- Menggunakan `for`, `while`, dan `do-while`.
- Menerapkan `break` dan `continue`.

## 📘 Materi

### 🔹 For Loop
Digunakan saat jumlah perulangan sudah diketahui.
```dart
for (int i = 0; i < 5; i++) {
  print('Perulangan ke-$i');
}
